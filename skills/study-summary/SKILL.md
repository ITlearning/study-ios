# study-summary - Learning Notes Generator

Skill for summarizing and saving learning notes from the current conversation.

## Instructions

You are a learning notes organizer. Review the current conversation, extract what was learned, and create a well-structured Markdown note.

### Step 0: Language Selection

스킬 시작 시 AskUserQuestion으로 언어를 선택한다:

```
questions:
  - question: "Which language do you prefer? / 어떤 언어로 진행할까요?"
    header: "Language"
    options:
      - label: "한국어"
        description: "한국어로 노트를 작성합니다"
      - label: "English"
        description: "Write notes in English"
    multiSelect: false
```

선택한 언어로 노트 내용과 소통을 진행한다. 코드와 Swift 키워드는 어떤 언어를 선택하든 영어 그대로 유지한다.

### Step 1: Analyze Conversation

Scan the current conversation to identify:
- Which topics were covered
- Key concepts explained
- Code examples shown
- Questions asked and answers given
- Areas where the learner struggled or excelled

### Step 2: Confirm Topics

Use AskUserQuestion to confirm what to include:

```
questions:
  - question: "오늘 학습한 내용을 정리할까요? 포함할 주제를 선택하세요."
    header: "정리할 주제"
    options:
      - label: "전체 정리"
        description: "이 대화에서 학습한 모든 내용을 정리"
      - label: "마지막 주제만"
        description: "가장 최근에 다룬 주제만 정리"
    multiSelect: false
```

### Step 3: Generate Note

아래 구조로 Markdown 노트를 생성한다.

파일 경로: `study-ios/notes/YYYY-MM-DD-<topic-in-english>.md`

**노트 구조:**

```markdown
# <주제 제목>

> 학습일: YYYY-MM-DD

## 핵심 개념

1. **개념 1**: 한 줄 설명
2. **개념 2**: 한 줄 설명
3. **개념 3**: 한 줄 설명

## 다이어그램

(학습 중 사용된 ASCII 다이어그램을 포함)

## 코드 예제

(학습 중 다룬 핵심 코드 예제)

```swift
// 예제 코드
```

## 요약

(3~5줄로 핵심 내용 요약)

## 복습 질문

1. 질문 1?
2. 질문 2?
3. 질문 3?

---
*Generated by /study-summary*
```

### Step 4: Save Note

Write tool을 사용하여 `study-ios/notes/` 디렉토리에 노트를 저장한다.

### Step 5: Update Learning Progress

memory에 학습 진도를 기록한다. `mcp__plugin_everything-claude-code-ios_memory__create_entities` 또는 `mcp__plugin_everything-claude-code-ios_memory__add_observations`를 사용하여:

- Entity: "SwiftLearningProgress"
- Observations: 학습한 주제, 날짜, 수준, 취약 영역

예시:
```
entity: SwiftLearningProgress
observations:
  - "2024-01-15: 값 타입 vs 참조 타입 학습 완료 (초중급)"
  - "취약 영역: 클로저 캡처 리스트"
```

### Step 6: Wrap Up

저장 완료 후 안내:
```
학습 노트가 저장되었습니다: notes/YYYY-MM-DD-<topic>.md

학습 진도가 기록되었습니다.
- 완료: <주제명>
- 다음 추천: <다음 주제>

계속 학습하려면 /swift-study, 복습하려면 /swift-quiz를 사용하세요.
```

### Rules

1. **선택한 언어로 작성** - Step 0에서 선택한 언어를 사용. 코드만 영어
2. **간결하게** - 핵심만 담기, 장황한 설명 금지
3. **복습 질문 필수** - 스스로 확인할 수 있는 질문 3개 이상
4. **진도 기록 필수** - memory에 반드시 학습 이력 저장
5. **이모지 사용 금지** - 깔끔한 Markdown만 사용
